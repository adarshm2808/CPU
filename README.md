# CPU
This project focuses on the design and 3D modeling of a custom gaming PC build using SolidWorks. The model demonstrates internal hardware arrangement, airflow-oriented cabinet structure, and mechanical integration of high-performance computer components.  

## Overview

This project focuses on the design and 3D modeling of a custom gaming PC build using SolidWorks. The model demonstrates internal hardware arrangement, airflow-oriented cabinet structure, and mechanical integration of high-performance computer components.

The project was developed to understand enclosure design, component placement, cooling optimization, and assembly planning for modern PC systems.


## Features

* Complete custom PC chassis assembly
* Internal component layout design
* Airflow-oriented cooling arrangement
* GPU and motherboard clearance optimization
* Cable management pathway planning
* Realistic rendered CAD visualization

---

## Components Modeled

* PC Cabinet/Chassis
* Motherboard Layout
* GPU Mounting Area
* Cooling Fan System
* Power Supply Section
* Storage Drive Bays
* Side Panel Structure


## Software Used

* SolidWorks


## Design Process

1. Designed the outer cabinet structure and dimensions.
2. Created internal mounting layout for PC hardware components.
3. Developed airflow-oriented fan placement system.
4. Modeled cable routing and support structures.
5. Performed component fit and clearance analysis.
6. Applied rendering and visualization techniques for presentation.


## Technical Skills Demonstrated

* 3D CAD Modeling
* Product Design
* Mechanical Assembly Design
* Parametric Modeling
* Engineering Visualization
* Structural Layout Planning


## Applications

* Gaming PC Systems
* Workstation Builds
* Custom Computer Cabinets
* Cooling and Airflow Design
* Electronics Enclosure Design


## Project Outcome

Successfully developed a custom gaming PC assembly model demonstrating enclosure design, hardware integration, cooling layout optimization, and CAD assembly techniques.
